# SESSION-12 — atom-memory

**Prerequisites:** SESSION-11 complete  
**Goal:** Fork agentscope-reme into atom-memory, wiring pgvector (long-term) and Redis (short-term).  
**Estimated time:** 1 day

---

## Tasks

1. **Inspect agentscope-reme** — document existing memory backends in `atom-memory/UPSTREAM_DIFF.md`.

2. **pgvector backend** (`atom-memory/src/atom_memory/backends/pgvector_backend.py`)
   ```python
   class PgvectorMemoryBackend:
       def store(self, agent_id: str, content: str, embedding: list[float], metadata: dict): ...
       def search(self, agent_id: str, query_embedding: list[float], top_k: int = 5) -> list[dict]: ...
       def delete(self, agent_id: str, memory_id: str): ...
       def clear(self, agent_id: str): ...
   ```
   - `store`: INSERT into `memory_vectors`.
   - `search`: `SELECT ... ORDER BY embedding <=> $1 LIMIT $top_k` (cosine distance).
   - Embedding is generated by calling atom-llm's embeddings endpoint via GATE.

3. **Redis backend** (`atom-memory/src/atom_memory/backends/redis_backend.py`)
   ```python
   class RedisMemoryBackend:
       def store(self, agent_id: str, key: str, value: Any, ttl_s: int): ...
       def retrieve(self, agent_id: str, key: str) -> Any: ...
       def clear(self, agent_id: str): ...
   ```
   Keys namespaced as `atom:memory:{agent_id}:{key}`.

4. **Memory manager** (`atom-memory/src/atom_memory/manager.py`)
   - Reads `memory_config` for the agent from Postgres on init.
   - Routes short-term ops → Redis, long-term ops → pgvector.
   - Exposes a clean API: `remember(content)`, `recall(query, top_k=5)`, `forget(id)`.

5. **Integration with atom-sdk**  
   Update `AtomModelWrapper` to optionally attach a memory manager and inject relevant
   memories into the system prompt before each LLM call.

6. **Memory config from studio** — atom-runtime reads `memory_config_id` from agent record
   and injects config as env vars into the agent pod.

---

## Acceptance Criteria

- [ ] `remember("The customer's credit limit is 50000")` stores a vector in Postgres.
- [ ] `recall("credit limit")` returns the stored memory with similarity > 0.8.
- [ ] Short-term Redis entries expire after the configured TTL.
- [ ] `clear(agent_id)` removes all memory entries for that agent.
- [ ] Memories are scoped per agent (agent A cannot access agent B's memories via this API).
